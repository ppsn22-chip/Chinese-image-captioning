## 关于服务器端的说明

服务器端需要搭建一个PHP环境,也就是常用的LAMP架构,则将PHP脚本文件直接放在 var/www/html 文件夹下即可,可通过IP+文件名的方法直接访问并执行脚本.
本项目的服务器代码将在后期测试完善后补全.

需要注意的是,存储上传文件的地方需要有777权限.
